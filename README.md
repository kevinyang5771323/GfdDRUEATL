# Python学生管理系统 python276

## 项目概述

Python学生管理系统是一个现代Web技术栈构建的全功能教育管理平台。它采用前后端分离的架构，提供包括学生信息管理、成绩管理、教师管理等核心功能，为学校教育管理工作提供便捷高效的解决方案。

## 技术栈

### 前端技术栈
- 框架: Vue 3 + Composition API
- 构建工具: Vite
- UI组件库: Element Plus
- 状态管理: Pinia
- 路由管理: Vue Router
- HTTP客户端: Axios
- 样式预处理: Sass/SCSS

### 后端技术栈
- 框架: Flask + SQLAlchemy
- 数据库: PostgreSQL（可配置Mysql5.7）
- 缓存: Redis
- 任务队列: Celery
- 认证: JWT (Flask-JWT-Extended)
- API文档: 自动生成的RESTful API
- 数据迁移: Flask-Migrate

## 功能模块

### 用户管理系统
- 多角色支持: 管理员、教师、学生
- 权限控制: 基于角色的访问控制(RBAC)
- 用户认证: JWT令牌认证
- 密码管理: 安全的密码加密和重置

### 学生管理
- 批量导入导出: 支持Excel/CSV格式
- 高级搜索: 多条件筛选和关键词搜索
- 数据统计: 学生分布统计和图表展示

### 教师管理
- 教师档案: 个人信息、职称、专业领域
- 部门管理: 学院、系部组织架构
- 异步导入: 后台任务处理大批量数据
- 权限分配: 灵活的教师权限配置

### 班级管理
- 班级信息: 班级基本信息、班主任、教室
- 专业管理: 多专业、多年级支持
- 学生分配: 班级学生关联管理
- 统计报表: 班级人数和分布统计

### 成绩管理
- 成绩录入: 支持单个和批量录入
- 成绩分析: 统计分析和图表展示
- 等级评定: 自动成绩等级计算
- 导出功能: 多格式成绩单导出

### 系统管理
- 角色权限: 细粒度权限控制
- 系统监控: 性能监控和日志管理
- 数据备份: 自动化数据备份策略
- 配置管理: 系统参数配置

### 任务管理
- 异步任务: Celery分布式任务队列
- 进度跟踪: 实时任务进度监控
- 错误处理: 完善的错误处理和重试机制
- 任务历史: 任务执行历史和结果查看

## 运行环境

- PostgreSQL 15.3（可配置Mysql5.7）
- Redis 5.0.4
